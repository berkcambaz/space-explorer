<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* pnpm
  ```sh
  npm install -g pnpm
  ```
  

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/berkcambaz/space-explorer.git
   ```
2. Change directory to /web
   ```sh
   cd /web
   ```
3. Install PNPM packages
   ```sh
   pnpm install
   ```
4. Run the project & check localhost:5173
   ```sh
   pnpm run dev
   ```
